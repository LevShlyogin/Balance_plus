# Данные для валидации расчетного комплекса [Название]

В этой директории хранятся входные файлы и эталонные результаты для проведения валидации расчетного ядра.

## Назначение
Цель этих данных — подтвердить точность расчетов комплекса путем сравнения с текущими результатами в системе
